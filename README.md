STUDENT PERFORMANCE PREDICTOR
 
Live App: https://student-performance-predictor-tcg4zxjwstftaelid7n2gi.streamlit.app/

 Overview

This project predicts student marks based on study habits such as hours studied, attendance, and sleep.


Features

* Data analysis using Pandas
* Machine learning model using Linear Regression
* Interactive web app using Streamlit

 Tech Stack

* Python
* Pandas
* Scikit-learn
* Streamlit
* Matplotlib


Model Performance

* Mean Squared Error: ~1.94

How to Run

pip install -r requirements.txt
streamlit run app.py

 Key Insights

* Study hours have the strongest impact on marks
* Attendance has moderate influence
* Sleep has smaller influence
